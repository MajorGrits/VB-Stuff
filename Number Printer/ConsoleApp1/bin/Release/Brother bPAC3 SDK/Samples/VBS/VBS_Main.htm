VB Script Samples (for Internet Explorer)

VBS_NamePlate

Summary

HTML page for creating nameplates

Enter the name and department to create nameplate labels. One of two templates (simple and framed) can be selected for making the nameplate.

Methods used

Open(), GetObject(), Text[Property], StartPrint(), PrintOut(),
EndPrint(), Export(), Close()

VBS_ChangeImage

Summary

HTML page for creating labels containing images

Specify the image and enter the name to create the label. Optional bitmap files can be specified for the graphics.
A character can specify an optional bitmap file.

Methods used

Open(), GetObject(), SetData(), Text[Property],
StartPrint(), PrintOut(), EndPrint(), Export(), Close()

VBS_DB_Bcd

Summary

HTML page for accessing a database and printing labels containing the barcode data and name for the specified item

First, create a Microsoft® Access MDB file corresponding to the items and barcode data. Access the database via ADO, retrieve the barcode data corresponding to the item, replace the barcode, and then print.

Methods used

Open(), GetObject(), Text[Property], SetData(), StartPrint(),
PrintOut(), EndPrint(), Export(), Close()

Note

In order to access the database using ADO, the MDAC component (mdac_typ.exe) must first be installed.

(This component is already installed on Windows® 2000 and the Millennium Edition.)
